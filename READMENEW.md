# דאשבורד אנליזת נוסעי הטיטאניק

דאשבורד אינטראקטיבי לניתוח נתוני הנוסעים של אוניית הטיטאניק, בנוי עם Streamlit. הממשק בעברית ומאפשר סינון נתונים והצגת תובנות ויזואליות.

## תכונות

- **סינון לפי מגדר** — בחירת נוסעים לפי `Sex`
- **סינון לפי מחלקה** — בחירת מחלקת נסיעה (`Pclass`)
- **טבלת נתונים** — הצגת רשומות הנוסעים לאחר הסינון
- **שיעור שרדות לפי מגדר** — גרף עמודות של אחוזי השרדות
- **פיזור גילאים** — היסטוגרמה של גילאי הנוסעים לפי סטטוס שרדות
- **שיעור שרדות לפי מחלקה** — גרף עמודות לפי מחלקת הנסיעה

## דרישות מערכת

- Python 3.8 ומעלה
- [Streamlit](https://streamlit.io/)
- [Pandas](https://pandas.pydata.org/)
- [Plotly](https://plotly.com/python/)

## התקנה

1. שכפול המאגר:

   ```bash
   git clone https://github.com/JonathanZouari/python.git
   cd python
   ```

2. התקנת התלויות:

   ```bash
   pip install streamlit pandas plotly
   ```

## הפעלה

הרצת הדאשבורד:

```bash
streamlit run titanic_dasboard.py
```

האפליקציה תיפתח בדפדפן (בדרך כלל בכתובת `http://localhost:8501`).

## מקור הנתונים

הנתונים נטענים ממאגר הטיטאניק הציבורי ב-GitHub:

`https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv`

לאחר הטעינה הראשונה, הנתונים נשמרים במטמון מקומי לטעינה מהירה יותר.
